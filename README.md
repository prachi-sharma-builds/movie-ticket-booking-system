#  Movie Ticket Booking System (Python)

A console-based Movie Ticket Booking System built using Python that simulates the basic workflow of booking movie tickets.

This project helped me practice Object-Oriented Programming (OOP) concepts by building a small real-world application.


##  Features

- Display movie details
- Book movie tickets
- Add extra seats after booking
- Automatic ticket cost calculation
- Display customer booking summary
- Processing simulation using Python's `time` module



## Concepts Used

- Python Classes & Objects
- Inheritance
- Constructors (`__init__`)
- Methods
- Variables & Attributes
- `super()`
- Time Module
- Basic Cost Calculations
